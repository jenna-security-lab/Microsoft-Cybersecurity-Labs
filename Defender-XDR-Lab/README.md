# Microsoft Defender XDR Lab
## Objective
Build a simulated enterprise environment endpoint security environment using Microsoft Defender XDR to practice security monitoring, investigation, and incident response.

## Environment

- Windows Server 2022 Domain Controller
- Windows 11 endpoints
- Microsoft Entra ID
- Microsoft Defender XDR

## Skills Demonstrated

- Endpoint onboarding
- Security policy configuration
- Alert investigation
- Threat hunting
- KQL
- MITRE Att&CK mapping

## Scenarios Completed

- Malware detection
- Phishing investigation
- Suspicious PowerShell activity
- Account security investigation
