|  Security Data Flow  |
|---|
|  1. User endpoints authenticate through Active Directory.  |
|  2. Windows endpoints generate security telemetry.  |
|  3. Microsoft Defender sensors collect endpoint activity.  |
|  4. Defender XDR analyzes processes, files, network connections, user activity, and security events  | 
|  5. Security alerts are investigated through the Defender portal.  |
|  6. Analysts perform threat hunting using KQL queries.  |
